## 项目介绍

> 本项目为跟随b站小满zs做的vue3项目，项目背景是做一个可视化的疫情查看图表，后台数据是根据`腾讯新闻疫情`使用express取得数据

> 因为只有一个页面，所以数据代码都写在入口文件App.vue中

## 技术栈

> Vue3 + vite + Ts + pinia

## 启动方法

```sh
npm install
```

```sh
npm run dev
```


```sh
npm run build
```
